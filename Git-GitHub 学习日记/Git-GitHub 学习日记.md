## Git/GitHub 学习指南

### 一. GitHub 使用指南

- GitHub 首页（小猫)
- 个人主页（profile）
- 仓库主页（repositries）
- 收藏（Star）
- 关注（Watch）
- 克隆项目（Fork）
- 事务卡片讨论（Issue）
- 发起修改请求（Pull Request）--需要先克隆项目进行改进

### 二. gitHub 仓库操作指南

- 新建仓库
- 新建文件
- 编辑文件
- 删除文件
- 搜索仓库文件
- 下载（zip格式，不怎么使用）
- 删除仓库（验证）
